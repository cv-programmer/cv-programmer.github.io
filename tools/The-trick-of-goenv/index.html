goenv使用说明

发表于2023-08-13|更新于2024-08-31|工具
|阅读量:

说明

  • 管理多版本Go环境

使用

  1. 安装：git clone https://github.com/go-nv/goenv.git ~/.goenv

  2. 添加以下内容至环境变量，如~/.bashrc，并使配置生效source ~/.bashrc

  3. 安装Go不同版本，如goenv install 1.20.6

  4. 查看已有版本：goenv versions

  5. 切换全局版本：goenv global 1.20.6

  6. 新环境二进制命令全局生效（goenv global后或者go install后）：goenv rehash
