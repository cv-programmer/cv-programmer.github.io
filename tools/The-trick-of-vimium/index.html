在 Chrome 中使用 vim

发表于2024-04-08|更新于2024-08-31|工具
|阅读量:

常用命令

  • 基本使用与 vim 相同，可见 vim使用说明
  • 安装后，快捷键优先级比网站自带命令高，比如 Github 全局搜索 /会被 vimium 占用
  • 小写字母表示当前页面操作，大写字母表示打开新页面操作
  • 官方文档可见 Vimium - The Hacker’s Browser

当前页面导航

  • ?: 显示命令
  • i: 插入模式，使所有命令失效，esc后退出
  • /: 页面内搜索
  • r: 刷新页面
  • yy: 粘贴当前页面 URL
  • p: 在当前页面打开剪贴板 URL
  • P：在新页面打开剪贴板 URL
  • gs: 查看网页源代码
  • f: 显示页面内链接快捷键，并在当前页面打开
  • F: 显示页面内链接快捷键，并在新页面打开
  • t: 创建新 tab

新页面导航

  • o: 当前页面打开 标签，书签及历史记录
  • O: 新页面打开 标签，书签及历史记录
  • b: 当前页面打开 书签
  • B: 新页面打开 书签

标签页间操作

  • J: 上一个 tab
  • K: 下一个 tab
  • t: 创建新 tab
  • x: 关闭当前 tab
  • X: 打开关闭的 tab
  • yt: 复制当前 tab

参考

  • https://github.com/philc/vimium/blob/master/README.md

  • 让你用 Chrome 上网快到想哭：Vimium
