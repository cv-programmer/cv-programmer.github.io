【Mongo】01-基础命令

发表于2023-10-19|更新于2024-08-31|数据库
|阅读量:

基础命令

  • 命令提示：help
  • 数据库方法帮助信息：db.help()
  • 查看数据库：show dbs
  • 进入数据库：use [数据库名]
  • 查看数据库表/集合：show tables /show collections
  • 查看数据：db.表名/集合名字.find()

容器

  • 使用镜像: bitnami/mongodb:7.0-debian-11

  • 进入MondoDB容器中

    • docker exec -it [容器名] mongosh
    • 如: docker exec -it mongo-test mongosh

  • 使用容器连接指定地址数据库

    • docker run -it docker.io/bitnami/mongodb:6.0-debian-11 mongosh [地址]
    • 如: docker run -it docker.io/bitnami/mongodb:6.0-debian-11 mongosh mongodb://mongodb0.example.com:27017,mongodb1.example.com:27017,mongodb2.example.com:27017/mydbname
