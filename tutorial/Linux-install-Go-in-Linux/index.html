【Linux】Go安装

发表于2022-11-07|更新于2024-08-31|教程
|阅读量:

说明

  • 环境：Windows子系统WSL并安装zsh，原生Linux安装步骤类似

步骤

  1. 检查是否已经安装过Go，输入go version，如果出现以下类似内容，说明未安装过

  2. 去官网下载需要的Go版本，此次安装版本为1.19.3，即下图中红框所示部分，可通过命令下载wget https://golang.google.cn/dl/go1.19.3.linux-amd64.tar.gz

  3. 将压缩包解压到/usr/local下：sudo tar -zxvf go1.19.3.linux-amd64.tar.gz -C /usr/local

  4. 将/usr/local/go/bin添加到环境变量中，由于本例用的是zsh，故修改的是~/.zshrc，如果是bash，则修改~/.bashrc
