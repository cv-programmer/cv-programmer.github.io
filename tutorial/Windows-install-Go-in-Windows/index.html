【Windows】Go安装

发表于2022-11-07|更新于2024-09-24|教程
|阅读量:

说明

以下演示为通过安装包安装，也可下载压缩版，解压缩再配置环境变量即可

环境变量新增 GOROOT 和 GOPATH，GOROOT 为 go 解压缩路径，GOPATH 任意设置路径即可

步骤

  1. 检查是否已经安装过Go，打开命令行（Win + R后输入cmd回车），运行go version，如果出现以下界面，说明未安装过

  2. 去官网下载需要的Go版本，此次安装版本为1.19.3，即下图中红框所示部分

  3. 下载完成后，双击运行，出现以下界面，根据提示安装即可

  4. 重新打开一个命令行，再次输入go version，可以看到，已经安装成功

  5. 设置国内代理
