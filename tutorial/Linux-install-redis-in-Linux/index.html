【Linux】redis安装

发表于2022-12-23|更新于2024-08-31|教程
|阅读量:

说明

  • 以centos源码编译安装为例

步骤

  1. 下载压缩包，Redis官网，下载版本为 7.0.3：wget https://github.com/redis/redis/archive/7.0.3.tar.gz

  2. 解压缩并移动到 /usr/local/redis，进入该目录

  3. 安装redis

  4. 启动redis

  5. 其他：redis-cli是连接本地redis服务的一个命令，通过该命令后可以即可控制redis的脚本控制台（执行脚本都在 bin/下）
