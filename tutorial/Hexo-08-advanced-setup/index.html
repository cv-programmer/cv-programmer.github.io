【Hexo】08-高级设置

发表于2022-12-14|更新于2024-08-31|教程
|阅读量:

文章跳转

文章内跳转

  • 锚点 [文章内跳转](#目标位置)
  • 目标位置 <span id="目标位置"></span>

不同文章跳转

  • 格式：{% post_link 文章文件名（不要后缀） 文章标题（可选） %}

  • 注

    • 文章标题不用写，会自动解析

    • 不需要带后缀.md

    • 要跳转文章md文件名都在_posts文件夹下

    • 如果_posts下有多级目录，如tools，相应要变成{% post_link tools/要跳转文章md文件名 %}，即根目录是_posts，其余子文件夹需要注明

    • 存在问题：本地跳转不友好，后面看能不能解决

搜索

  1. 安装hexo-generator-search：npm install hexo-generator-search --save

  2. 修改主题配置文件，如修改_config.butterfly.yml，默认只有enable改成true即可
