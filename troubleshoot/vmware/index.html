【TroubleShooting】Vmware 相关

发表于2024-07-21|更新于2024-08-31|troubleshoot
|阅读量:

说明

  • 以下环境无特别说明，均在 VMware® Workstation 17 Pro 17.5.2 build-23775571 虚拟机发生

centos7-mini

说明

  • 系统为 CentOS-7-x86_64-Minimal-2009.iso

基础环境

  • ifconfig: yum install net-tools

非 root 每次执行需要输入 sudo

  • 修改 /etc/sudoers，加入以下内容，god 是用户名 => root 用户改了，没生效，但是 god 用户改了，生效？

ip addr 无 ipv4 地址

  • 修改 /etc/sysconfig/network-scripts/ifcfg-ens33，将 ONBOOT NO 修改为 NOBOOT YES，重启虚拟机后，即有 ipv4 地址
  • 具体修改文件，以显示的网卡为准，不一定为 ifcfg-ens33
  • 参考 centos7的yum命令无法使用解决方法

无法使用 yum 安装

  • 报错如下

  • 修改 /etc/yum.repos.d/CentOS-Base.repo，配置为国内源（快捷替换为 aliyun 源: curl -o /etc/yum.repos.d/CentOS-Base.repo https://mirrors.aliyun.com/repo/Centos-7.repo）

  • 配置生效
